# OrzLarkSDK

Lark SDK write by swift programming language


## Feature

- send message to Lark Group with a webhook which fetched from the custom bot that been added into Lark Group

- [support message type][message type]:
    
    - text
    
    - post
    
    - image
    
    - share_chat
    
    - interactive
